This folder contains the contents of a swimming times tracker! I used to be a competitive swimmer so maybe this will come in handy to whoever swims. I made this program for fun, and to also get more familiar with reading and writing to files with python.

The program first asks the user to etiher enter times or view existing times.

Existing times:
If the user wishes to view times, the user selects view times and then is prompted to view what stroke they need to view (butterfly, backstroke, breaststroke, freestyle, or IM). 
The user selects the stroke and the user can view the events, times, and the dates they occurred.


Entering new times:
If the user wants to enter times to the files, the user is first prompted with a menu that lists all strokes (butterfly, backstroke, breaststroke, freestyle, or IM). Then the userr is prompted what even they would like to enter times for Ex: 200 freestyle.
The user enters the time for that event, then the user is confirmed that the data was written to the designated file for that stroke. The user is then asked to enter more times. If the user enters "y" or "Y", then the programs jumps bak to select what stroke they would like to enter times for.
If the user enters "n" or "N", then the program says "See you after your next swim!" and the program shuts down. If the user enters a different value other than "y" , "Y" , "n" , or "N", then the user gets a different message and the program shuts down.

Hope you enjoy!
