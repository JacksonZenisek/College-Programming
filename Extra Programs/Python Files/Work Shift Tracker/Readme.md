The company that I work for has four shifts: A shift, B shift, C Shift, and D shift.

This program displays the current working shift based on this schedule:

<p><b>
Short Week:
Sunday - Tuesday 6:00am = 6:00pm = A shift
Sunday - Tuesday 6:00pm = 6:00am = C shift
Wednesday - Saturday 6:00am = 6:00pm = B shift
Wednesday - Saturday 6:00pm = 6:00am = D shift

Long Week:

Sunday - Tuesday 6:00am = 6:00pm = A shift
Sunday - Tuesday 6:00pm = 6:00am = C shift
Wednesday - Saturday 6:00am = 6:00pm = B shift
Wednesday - Saturday 6:00pm = 6:00am = D shift
</b>
</p>

<p><b>Note: Short weeks are weeks that A and C shift don't work on Wednesdays. Long weeks are weeks that A and C shift do work on Wednesdays.</b></p>

The program also displays the day of the week, the date, and the time (12 hr clock) of when the program was excecuted to help reference the current shift. It also displays the current week schedule for all four shifts based it either being a
"short week" or a "long week". I converted this python file to a .exe file, which I now frequently use to know what shift is currelty working.
